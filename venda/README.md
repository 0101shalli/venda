# venda
Venda is a secure, cross-platform desktop management system tailored for general retail stores. Built using React, Python, and SQLite, it offers role-based dashboards for admins, managers, and cashiers. Venda handles inventory tracking, barcode scanning, thermal printing, and comprehensive sales analytics with ease.

## Run the system

1. Open a terminal and navigate to the project backend folder:

   ```bash
   cd /home/lucas/venda_V1/venda
   ```

2. Make sure the script is executable (only required once):

   ```bash
   chmod +x run.sh
   ```

3. Start the frontend and backend together:

   ```bash
   ./run.sh
   ```

4. Open the app in your browser:

   - Frontend: `http://localhost:5173`
   - Backend API: `http://localhost:8000`

5. Stop the system by pressing `Ctrl+C` in the terminal.
